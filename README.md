# 🧮 Calculadora Tributária em Excel

Esta planilha foi criada para **automatizar o cálculo de DIFAL (Diferencial de Alíquota) e Substituição Tributária (ST)** com base na leitura de arquivos XML de NFe (Nota Fiscal eletrônica). Ela facilita o preenchimento de valores para GNRE e reduz erros manuais no processo.

---

## ✅ Funcionalidades

- Importação de XML
- Cálculo automático de tributos estaduais
- Fórmulas com validações para reduzir erros
- Layout fácil de usar no dia a dia fiscal

---

## 🛠️ Ferramentas utilizadas

- **Excel Avançado** (fórmulas, validações e lógica aplicada)
- Conhecimento em regras fiscais (ICMS, DIFAL, ST)
